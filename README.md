# Ember

A dark theme with a warm, amber-toned palette.

> **Note:** This is a personal project, very early in development. It includes themes for the tools I use, configured the way I like them. Your mileage may vary.

## Supported

- Neovim
- Zed
- Ghostty
- Firefox (via [Firefox Color](https://addons.mozilla.org/en-GB/firefox/addon/firefox-color/))
- OpenCode
- Mako
- Tofi
- Ashell

## Colors

| Role     | Hex       | Usage                    |
|----------|-----------|--------------------------|
| yellow   | `#f0c000` | Functions                |
| orange   | `#e88a2a` | Keywords                 |
| red      | `#d44a3a` | Constants, numbers       |
| green    | `#a9b665` | Strings                  |
| blue     | `#7aa2c7` | Types                    |
| tan      | `#c4a87a` | Properties               |

See [DESIGN.md](DESIGN.md) for the full color palette and design decisions.
